# aritra-maxxing 🧭

A layered **junior-year operating console** for Aritra Bhattacharya — UGA Terry College **Management Information Systems ('27)**, minoring in mathematics & astrophysics.

One page, lots of layers: the two mandatory professional-fraternity rushes (**Delta Sigma Pi** + **Alpha Kappa Psi**), a club portfolio, a Summer-2027 recruiting pipeline, a skills matrix, and a weekly operating rhythm — all as clean, scannable tables.

## View it

- **Live page:** open `index.html` in any browser, or enable GitHub Pages (Settings → Pages → Deploy from `main` / root).
- Single self-contained file — no build step, no dependencies.

## What's inside

| Layer | What it does |
|---|---|
| Scoreboard | The whole plan in six KPIs |
| Trajectory | Four phases from summer sprint → landed internship |
| Rush tracker | DSP + AKΨ (+ Beta Alpha Psi) rounds, requirements, next actions |
| Portfolio | UGA clubs & orgs ranked by MIS leverage |
| Pipeline | Role tracks + employers that recruit Terry |
| Loadout | Skills matrix — each ending in a shippable proof |
| Cadence | A themed-day weekly rhythm |
| Doctrine | Rules of the grind |
